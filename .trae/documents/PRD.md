# 股票 AI 分析 Agent - 产品需求文档

## 1. 产品概述

**产品名称：** Stock Analyst Agent 前端
**产品类型：** Web 应用程序
**核心功能：** 提供图形化界面操作股票 AI 分析 Agent，支持 K 线数据获取、新闻搜索、AI 分析和报告展示
**目标用户：** 股票投资者、量化交易者、金融分析师

## 2. 用户需求

### 2.1 核心功能需求

#### F1: 股票搜索与分析
- 用户输入股票代码和名称
- 选择市场类型（A股、港股、美股）
- 选择分析周期（日线、周线、月线）
- 设置分析天数
- 发起分析请求

#### F2: K线数据展示
- 以表格形式展示 K 线数据
- 显示关键技术指标（MA5、MA10、涨跌幅等）
- 支持数据的视觉化呈现

#### F3: 新闻资讯展示
- 展示股票相关新闻列表
- 显示新闻标题、来源、时间
- 支持情感分析结果展示

#### F4: AI 分析报告
- 展示技术面分析结果
- 展示消息面分析结果
- 展示投资建议和风险等级
- 支持报告导出

#### F5: 配置管理
- 配置 AI 接口（API Key、模型选择）
- 配置数据源
- 管理分析参数

### 2.2 用户交互流程

```
1. 用户打开网页
   ↓
2. 进入分析页面
   ↓
3. 输入股票代码和名称
   ↓
4. 选择市场、周期等参数
   ↓
5. 点击"开始分析"按钮
   ↓
6. 显示加载状态
   ↓
7. 展示 K 线数据
   ↓
8. 展示新闻资讯
   ↓
9. 展示 AI 分析结果
   ↓
10. 用户可导出报告或继续分析
```

## 3. 功能详细设计

### 3.1 页面结构

| 页面 | 功能 | 路由 |
|------|------|------|
| 首页/分析页 | 股票搜索、分析、结果展示 | `/` |
| 配置页 | AI 配置、数据源配置 | `/settings` |
| 历史页 | 查看历史分析记录 | `/history` |

### 3.2 分析页组件

| 组件 | 功能 |
|------|------|
| StockSearchForm | 股票搜索表单 |
| MarketSelector | 市场选择器 |
| PeriodSelector | 周期选择器 |
| KLineChart | K 线图表展示 |
| KLineTable | K 线数据表格 |
| NewsList | 新闻列表 |
| TechnicalIndicators | 技术指标卡片 |
| AIAnalysisResult | AI 分析结果展示 |
| LoadingOverlay | 加载遮罩 |

### 3.3 API 接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/analyze` | POST | 分析单只股票 |
| `/api/kline` | GET | 获取 K 线数据 |
| `/api/news` | GET | 获取新闻数据 |
| `/api/config` | GET/POST | 获取/设置配置 |

## 4. 验收标准

- [ ] 用户可输入股票代码和名称进行分析
- [ ] 支持选择不同市场类型
- [ ] K 线数据正确展示
- [ ] 新闻资讯正确展示
- [ ] AI 分析结果正确展示
- [ ] 界面美观、响应流畅
- [ ] 支持移动端适配
- [ ] 配置页面可管理 AI 设置
- [ ] 报告可导出为文本/JSON 格式
