# 股票 AI 分析 Agent - 技术架构文档

## 1. 技术栈

### 1.1 前端
- **框架：** React 18 + TypeScript
- **构建工具：** Vite
- **样式：** Tailwind CSS
- **状态管理：** Zustand
- **HTTP 客户端：** Axios
- **图表库：** Recharts
- **图标：** Lucide React

### 1.2 后端
- **框架：** Express.js
- **语言：** TypeScript
- **Agent 逻辑：** Python（通过子进程调用）

## 3. API 设计

### 3.1 分析接口

**POST** `/api/analyze`

Request:
```json
{
  "stock_code": "600519",
  "stock_name": "贵州茅台",
  "market": "A股",
  "period": "日线",
  "days": 30
}
```

Response:
```json
{
  "stock_code": "600519",
  "stock_name": "贵州茅台",
  "analysis_date": "2024-01-15",
  "technical_analysis": {
    "trend": "上涨",
    "support_level": 1650.0,
    "resistance_level": 1800.0,
    "indicators_summary": "MA5:1720.5, MA10:1705.3"
  },
  "news_analysis": {
    "sentiment": "偏正面",
    "key_events": ["业绩预告", "机构评级"],
    "market_feedback": "投资者关注度高"
  },
  "recommendation": "买入",
  "risk_level": "中等",
  "summary": "综合分析总结"
}
```

### 3.2 K 线数据接口

**GET** `/api/kline?code=600519&market=A股&period=日线&days=30`

Response:
```json
{
  "data": [
    {
      "date": "2024-01-01",
      "open": 1700.0,
      "high": 1750.0,
      "low": 1680.0,
      "close": 1720.0,
      "volume": 5000000

    "ma5": 1720.5,
    "ma10": 1705.3,
    "ma20": 1680.0,
    "price_change": 3.5,
    "volume_change": 15.0
  }
}
```

### 3.3 新闻接口

**GET** `/api/news?code=600519&name=贵州茅台&days=7`

Response:
```json
{
  "data": [
    {
      "title": "贵州茅台发布年度业绩预告",
      "url": "https://...",
      "publish_time": "2024-01-15 10:30",
      "source": "东方财富",
      "summary": "..."
    }
  ],
  "sentiment": {
    "sentiment": "偏正面",
    "positive_count": 5,
    "negative_count": 1,
    "neutral_count": 4
  }
}
```

### 3.4 配置接口

**GET** `/api/config`

Response:
```json
{
  "ai": {
    "provider": "openai",
    "api_key": "***",
    "base_url": "",
    "model": "gpt-4"
  },
  "stock": {
    "default_market": "A股",
    "default_period": "日线",
    "default_days": 30
  }
}
```

**POST** `/api/config`
- 更新配置（仅允许更新非敏感字段）

## 5. 响应式设计

|------|------|
| Desktop (≥1024px) | 双列布局，侧边图表 |
| Tablet (768-1023px) | 单列布局，图表堆叠 |
| Mobile (<768px) | 全宽布局，垂直堆叠 |
